LionBars by nikolaydyankov

A free to use, lightweight jQuery plugin that makes scrollbars look like in OSX Lion.

Dependencies

jQuery

Install

1. Download LionBars. 2. Include the .js file right after jQuery, like this: 3. Include the .css file in your main .css file, like this: @include 'lionbars.css'; 4. Call the lionbars() function on elements of your choise. For example: 5. Optional: lionbars(color, showOnMouseOver, visibleBar, visibleBg); color: Either "light" or "dark". Choose wisely! Defaults to "dark". showOnMouseOver: When the mouse moves over the content element, the scrollbar is shown. Defaults to false. visibleBar: Scrollbar is always visible. Defaults to false. visibleBg: Full length scrollbar (or scrollbar background) is always visible. Defaults to false.
